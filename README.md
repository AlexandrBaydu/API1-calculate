# API1-calculate
Web-API по дисциплине "Цифровые возможности для бизнеса"
Название API: «Калькулятор стоимости»

Основное назначение веб-сервиса: расчёт стоимости товаров/услуг.

Краткое описание решаемой задачи: проводить расчёт стоимости требуемых товаров/услуг.

Основные функции:
- Получение на вход данных для обработки, вводимых пользователем для формирования запроса.
- Получение запроса на вычисление стоимости товаров/услуг.
- Вычисление суммы и вывод данных на интерфейс.

Архитектурные особенности (используемые технологии, библиотеки, базы данных и т. п.):
- Веб-сервис написан на языке C#.
- Три основных программных компонента: модели данных, контроллер (выполнение базовой функции сервиса), программа (запускает сервис в браузере).
- Сервис содержит один POST и один GET запрос. POST-запрос отправляет вводимые пользователем данные с клиента на сервер, GET запрос возвращает эти же данные на клиент, но вместе с новой строкой – информацией результата работы контроллера.

Развёртывание веб-сервиса:
 
  Веб-сервис запущен в swagger для проверки функционала
   
  Схемы POST и GET запросов:
  {
  productName	[...]
  price	[...]
  quantity	[...]
  }
  {
  productName	[...]
  price	[...]
  quantity	[...]
  totalPrice	[...]
  }
  
  Пример работы POST-запроса:
   
    Тело запроса (данные не введены):
    {
      "productName": "string",
      "price": 0,
      "quantity": 0
    }
     
    Выполнение запроса без данных, выполняется проверка строк данных:
    Response body
    Цена должна быть больше 0
     
    Новые данные в теле запроса:
    {
      "productName": "Мука, 1кг",
      "price": 438,
      "quantity": 12
    }
     
    Ответ на запрос получен, выведены имя товара, цена, количество и конечная стоимость:
    Response body
    
    {
      "productName": "Мука 1кг",
      "price": 438,
      "quantity": 12,
      "totalPrice": 5256
    }
  
  Пример работы GET-запроса:
   
    Поля для ввода данных:
    productName string(query)
    
    price number($double)(query)

    quantity integer($int32)(query)
     
    Выполнение пустого запроса:
    Error: Bad Request
    Response body
  
    {
      "type": "https://tools.ietf.org/html/rfc9110#section-15.5.1",
      "title": "One or more validation errors occurred.",
      "status": 400,
      "errors": {
        "productName": [
          "The productName field is required."
        ]
      },
      "traceId": "00-772edea57661ed3aa60e67615b7f4c9f-9eeb48b6ae5c692a-00"
    }
     
    Поля данных заполнены:
    productName string(Мука)
    
    price number($double)(40)

    quantity integer($int32)(11)
     
    Запрос выполнен корректно:
    	
    Response body
    Download
    {
      "productName": "Мука",
      "price": 40,
      "quantity": 11,
      "totalPrice": 440
    }
